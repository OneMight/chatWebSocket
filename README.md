# Description:
This application provides possibility to research posts from users.
To use this application locally you should:
1. Clone repository to your local machine
```
git clone https://github.com/OneMight/chatWebSocket.git
```
2. Open IDE and execute command in a root dir:
   ```
    npm install 
   ```
3. Start server. To launch server you should execute this commands:
   For dev:
```
npm run dev
```
For prod:
```
npm run build
npm run preview

```
